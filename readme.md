# 📐 高等数学计算器

一个支持自然语言输入的高等数学计算器，支持复数运算、三角函数、对数、根号等多种数学运算。

## ✨ 功能特性

- **自然语言输入**：支持中文命令，如"30开5次根号"、"2的10次方"
- **复数运算**：支持虚数单位 i，可计算欧拉公式 e^(iπ) = -1
- **三角函数**：sin、cos、tan（支持角度输入）
- **对数运算**：ln（自然对数）、log（常用对数）
- **根号运算**：平方根、立方根、N次方根
- **常数支持**：π（圆周率）、e（自然常数）、i（虚数单位）
- **绝对值**：支持负数取绝对值
- **百分数**：支持百分数输入

## 🚀 快速开始

### 在线使用

访问在线版本：[https://rpsuperstar.github.io/newcomer/](https://rpsuperstar.github.io/newcomer/)

### 本地运行

```bash
git clone https://github.com/rpsuperstar/newcomer.git
cd newcomer
python -m http.server 8000
```

然后在浏览器中打开 `http://localhost:8000`

## 📝 使用示例

| 输入 | 结果 |
|------|------|
| 30开5次根号 | 约1.9744 |
| 2的10次方 | 1024 |
| sin(90) | 1 |
| cos(60) | 0.5 |
| ln(10) | 约2.3026 |
| e的iπ次方 | -1 |
| 根号(-1) | i |
| 绝对值(-20) | 20 |
| 30%开3次根号 | 约0.6694 |

## 🔢 支持的运算符

- 基本运算：加、减、乘、除、余
- 幂运算：平方、立方、N次方
- 开方运算：平方根、立方根、N次方根
- 三角函数：sin、cos、tan
- 对数运算：ln、log
- 绝对值：绝对值
- 常数：π、e、i
